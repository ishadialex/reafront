PDF FILES FOLDER
================

Place your PDF files in this folder with the following names:
- pdf-1.pdf
- pdf-2.pdf
- pdf-3.pdf

These PDFs will be accessible through the Documents menu in the header.

Users can:
- Read the PDFs in the browser
- Print the PDFs
- Download the PDFs

To add or update a PDF:
1. Place the PDF file in this folder (public/pdfs/)
2. Name it according to the menu item (pdf-1.pdf, pdf-2.pdf, or pdf-3.pdf)
3. The PDF will automatically be available through the website
